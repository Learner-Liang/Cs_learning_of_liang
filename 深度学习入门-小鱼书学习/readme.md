## 神经网络

* soft函数防止溢出技巧：所有得出的值减去其最大值。
* ```
  for i in range(0, len(x), batch_size):
      x_batch = x[i:i+batch_size] # 极其优雅的切片：提取第 i 到 i+100 张图片
      y_batch = predict(network, x_batch)
      p = np.argmax(y_batch, axis=1) # 找到这100张图各自预测概率最大的索引
      accuracy_cnt += np.sum(p == t[i:i+batch_size]) # 统计这一批里答对了几道题
  ```

  为了防止同时计算任务量太大，使用切片操作，选择一百个图像练习。
