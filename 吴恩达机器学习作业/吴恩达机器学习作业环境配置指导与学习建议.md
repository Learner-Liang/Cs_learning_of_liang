这份文档将指引你如何从零开始配置本作业所需的所有 Python 环境依赖。我们在这些作业（如：神经网络、推荐系统等）中主要使用到了以下几个核心库：

- `numpy`：强大的数学与矩阵运算库。
- `pandas`：数据处理与分析工具。
- `tensorflow`：用于搭建和训练深度学习模型（如：神经网络）。
- `scikit-learn`：提供数据预处理（如特征缩放、划分训练集/测试集等）和机器学习算法。
- `tabulate`：用于在终端或 Notebook 中美观地打印表格数据。
- `jupyter`：用于运行 `.ipynb` 格式的交互式笔记本。

为了避免与其他项目产生包版本冲突，强烈建议使用 **Conda** 或 Python 内置的 **venv** 来创建一个独立的虚拟环境。

## 方式一：使用 Conda 配置环境 (推荐)

如果你还没有安装 [Anaconda](https://www.anaconda.com/products/distribution) 或 [Miniconda](https://docs.conda.io/en/latest/miniconda.html)，请先前往官网下载并安装。

1. **打开命令行终端（Windows 用户请打开 Anaconda Prompt 或 PowerShell）：**
2. **创建一个新的 Conda 虚拟环境**（我们将其命名为 `ml-coursera`，使用 Python 3.9 版本）：

   ```bash
   conda create -n ml-coursera python=3.9 -y
   ```
3. **激活刚刚创建的虚拟环境：**

   ```bash
   conda activate ml-coursera
   ```
4. **安装本作业所需的所有依赖包：**

   ```bash
   pip install numpy pandas tensorflow scikit-learn tabulate jupyter
   ```
5. **启动 Jupyter Notebook：**
   在终端中通过 `cd` 命令进入到你存放作业代码的目录，然后运行：

   ```bash
   jupyter notebook
   ```

   随后会自动在浏览器中弹出一个页面，你就可以点击 `.ipynb` 文件开始做作业了。

## 方式二：使用 Python 内置虚拟环境 (venv) 配置

如果你只安装了原生的 Python 环境，你可以使用 `venv` 来创建虚拟环境。

1. **打开命令行终端（如 Windows 的 cmd 或 PowerShell）：**
2. **在你的项目根目录下，创建一个名为 `.venv` 的虚拟环境：**

   ```bash
   python -m venv .venv
   ```
3. **激活虚拟环境：**

   - **Windows:**
     ```bash
     .venv\Scripts\activate
     ```
   - **macOS / Linux:**
     ```bash
     source .venv/bin/activate
     ```

   *(激活成功后，你的命令行提示符前面会出现 `(.venv)` 的字样)*
4. **升级 pip 并安装依赖包：**

   ```bash
   python -m pip install --upgrade pip
   pip install numpy pandas tensorflow scikit-learn tabulate jupyter
   ```
5. **启动 Jupyter Notebook：**
   同样地，确保在作业所在的目录下，运行：

   ```bash
   jupyter notebook
   ```

## 常见问题 (FAQ)

- **导入 TensorFlow 报错？**
  如果你在 Windows 下遇到由于本地缺少 C++ 运行库导致 TensorFlow 加载失败，请前往微软官网下载安装最新的 [Visual C++ Redistributable](https://docs.microsoft.com/zh-cn/cpp/windows/latest-supported-vc-redist)。
- **在 VS Code 中如何选择对应的环境？**
  如果你使用 VS Code 打开 `.ipynb` 文件，请在右上角点击选择 Kernel（内核），然后选择刚才创建的 `ml-coursera` (Conda) 或者是 `.venv` 环境作为解释器。
